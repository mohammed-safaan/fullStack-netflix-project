# Netflix-clone:
A clone of Netflix movie web app with an admin dashboard

## Technologies:
  #### Front-end:
  1. Reactjs
  2. React-bootstrap
  3. Redux-toolkit
  4. Scss
  
  #### Back-end:
  1. Nodejs
  2. Express.js
  3. Mongoose
  4. Stripe (payment)
  
  #### Database:
  1. MongoDb
  2. Firebase
  
## Features:
1. Responsive layout
2. Landing page
3. SignIn & signUp pages
4. Home page
5. User favourites page
6. Movies page (with search & filter)
7. Series page (with search & filter)
8. Watch page
9. Handeled subscription with Stripe
10. Commenting system for each movie
11. Admin dashboard
